About

This report is a small contribution towards safer Rust supply-chains in the long run. It leverares public information available in the open-source ecosystem to inquire about which open-source crates adopts the RFC 3691 and also which ones can have build artifacts reproduced.

This report is updated at least once a week, and always target the default available version for a crate. The source code for this automation is available on Github, and you can generate this report by yourself!
